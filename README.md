## Requirement ##

*   SDL2 DLL
    *   https://www.libsdl.org/

## Usage ##

    $ python test_draw.py
    $ python test_gfx.py
    $ python test_ttf.py ./GenShinGothic-Normal.ttf
    $ python test_image.py 256px-Globe.svg.png

*   ESC : Exit application
